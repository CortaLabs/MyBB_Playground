- **Version:** 1.0.0
- **Author:** Austin
- **Compatibility:** MyBB 18*
- **Type:** plugin
- **Visibility:** public

## Installation

1. Copy contents of `inc/` to your MyBB installation's `inc/` directory
2. Go to Admin CP > Configuration > Plugins
3. Find "Forge Config Test" and click Install
4. Configure settings as needed

## Files

- `inc/plugins/forge_test.php` - Main plugin file
- `inc/languages/english/forge_test.lang.php` - Language strings

- Templates are stored in database (use MCP tools to manage)
