# Scaffold Test

Testing the new disk-first template scaffolding

## Information

- **Codename:** scaffold_test
- **Version:** 1.0.0
- **Author:** Developer
- **Compatibility:** MyBB 18*
- **Type:** plugin
- **Visibility:** public

## Installation

1. Copy contents of `inc/` to your MyBB installation's `inc/` directory
2. Go to Admin CP > Configuration > Plugins
3. Find "Scaffold Test" and click Install
4. Configure settings as needed

## Files

- `inc/plugins/scaffold_test.php` - Main plugin file
- `inc/languages/english/scaffold_test.lang.php` - Language strings

- Templates are stored in database (use MCP tools to manage)
