Test plugin for verifying file tracking and deployment

- **Version:** 1.0.0
- **Author:** Developer

- **Type:** plugin
- **Visibility:** public

## Installation

1. Copy contents of `inc/` to your MyBB installation's `inc/` directory
2. Go to Admin CP > Configuration > Plugins
3. Find "Test Tracker Plugin" and click Install
4. Configure settings as needed

## Files

- `inc/plugins/test_tracker.php` - Main plugin file
- `inc/languages/english/test_tracker.lang.php` - Language strings

- Templates are stored in database (use MCP tools to manage)
