# Dice Roller

BBCode dice rolling with database tracking and post CSS effects for tabletop gaming

## Information

- **Codename:** dice_roller
- **Version:** 1.0.0
- **Author:** Corta Labs
- **Compatibility:** MyBB 18*
- **Type:** plugin
- **Visibility:** public

## Installation

1. Copy contents of `inc/` to your MyBB installation's `inc/` directory
2. Go to Admin CP > Configuration > Plugins
3. Find "Dice Roller" and click Install
4. Configure settings as needed

## Files

- `inc/plugins/dice_roller.php` - Main plugin file
- `inc/languages/english/dice_roller.lang.php` - Language strings

- Templates are stored in database (use MCP tools to manage)
