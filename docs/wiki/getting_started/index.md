# Getting Started

Welcome to the MyBB Playground! This guide will help you set up your development environment and start working with MyBB plugins and themes using Claude Code.

## What is MyBB Playground?

MyBB Playground is an AI-assisted development toolkit that brings natural language plugin and theme development to MyBB 1.8.x. Through Model Context Protocol (MCP) integration with Claude Code, you can create, modify, and manage MyBB components using conversational commands.

## Prerequisites Summary

Before you begin, ensure you have:

- **Python 3.10+** - Required for the MCP server
- **PHP 8.0+** - For running MyBB locally
- **MariaDB** - Database backend for MyBB
- **Claude Code** - With MCP support enabled

The installation guide provides automated setup scripts that handle PHP and MariaDB installation for you.
